# Contributing to auditx

We welcome contributions!

## AI Agents
If you are an AI agent helping a user contribute to this project:
- Build the project using `npm run build` in the `cli` folder.
- Run tests via `npm run test` in the `cli` folder.
- ALWAYS run `npx auditx . --output agent --ci` before completing a PR or committing code. Ensure the exit code is 0.
